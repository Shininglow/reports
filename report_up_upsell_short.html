Summary

  1. Primary metric (ARPU): v5 виграє, +14%. v4 $4.04 vs v5 $4.59. Bayesian P(v5>v4) = 91.1%, HDI [−$0.26, +$1.38] включає 0 — технічно не стат значимий на 95% рівні, але expected loss при шіпінгу v4 ($0.58/user) у 34 рази вищий ніж при шіпінгу v5 ($0.02/user).
  2. ARPU gain driven entirely by ARPPU. CR ідентична (18%, p=0.916). v5 ARPPU значно вищий (P = 97.2%) — повністю за рахунок вищої ціни long page ($29.99 vs $24.99). Conversion rate 45.8% — сигналу нема.
  3. Long page CR незначно покращився (+1.02 pp, 9.76% → 10.78%, p=0.268). Вищий прайс + comparison table не відлякав покупців.
  4. Short funnel CR незначно знизився (−1.18 pp, 9.23% → 8.05%, p=0.192). Всі різниці по каскаду — не стат значимі.
  5. Revenue концентрується на long page: частка зросла з 62% до 72% — підтверджує, що вищий прайс є основним драйвером.
  6. Гіпотеза підтвердилась: v5 delivers +14% ARPU (+16% revenue) при збереженні CR на рівні v4.
